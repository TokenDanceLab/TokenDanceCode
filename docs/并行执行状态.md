## Public Boundaries

- npm publish is not automated and must not be part of verification scripts.
- TokenDance Gateway model calls use TokenDance API keys. TokenDanceID/OIDC is for identity/session flows in consuming products.
- Public docs may name environment variables but must not include real keys, token-bearing config files, local operator paths, or production logs.
- Old Python files are retained as migration reference only. Current development targets the TypeScript packages.
