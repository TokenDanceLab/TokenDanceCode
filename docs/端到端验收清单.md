## 11. 故障排查验收

- [ ] `tokendance` 命令不可用：确认 editable install 或 pipx install 已完成，重新打开 PowerShell。
- [ ] PowerShell 激活失败：直接用 `.\.venv\Scripts\python.exe` 运行命令，不修改全局执行策略。
- [ ] 无 API key：设置 `$env:OPENAI_API_KEY` 或 `$env:ANTHROPIC_API_KEY`。
- [ ] resume 无 session：先启动一次 `tokendance` 并 `/exit`。
- [ ] Git 命令失败：确认当前目录是 Git 仓库。
- [ ] 权限被拒绝：确认目标路径在 workspace 内，且命令不是高风险 PowerShell 命令。
- [ ] task/todo 交互不可用：先确认 provider 是否发出了对应 tool call，再运行 service/tool/runtime registry 单元测试定位。
- [ ] subagent worktree 不可用：确认 Stage 13 文件、默认 runtime 工具注册与 slash 命令是否已经合并。
