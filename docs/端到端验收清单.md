# 端到端验收清单

本文档对应 `docs/开发流程文档.md` 的阶段 15：安装、文档与端到端验收。手动验收建议在一次性测试仓库或独立 worktree 中执行，避免污染正在开发的主工作区。

> TS 重构分支更新：当前权威验收先看本节。下面 Python 验收项保留为 v0.1 参考，直到对应功能迁移到 TypeScript 后再逐项替换。

## TS-0. 当前 TypeScript 闭环验收

- [x] 在 `codex/ts-refactor` worktree 中安装依赖。

```powershell
pnpm install
```

